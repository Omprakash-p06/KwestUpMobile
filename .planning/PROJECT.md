# KwestUp Mobile

## What This Is

KwestUp Mobile is a local-first, privacy-respecting personal organizer application designed to bring the organizational power of **Obsidian** and **Notion** directly to your phone. It seamlessly integrates rich markdown note-taking, a Google Tasks-style management interface, and a contact birthday tracker with dynamic alerts.

A key highlight of the app is its ability to synchronize locally over Wi-Fi with the **KwestUp PC** desktop app (an Electron utility) using a quick camera scan of a PC-generated QR code. To empower user productivity on-device, KwestUp Mobile hosts a fully offline local AI model (0.5B to 1B parameters) running native inference via `react-native-llama` to summarize notes and auto-extract actionable tasks without sending any data to the cloud.

---

## Core Value

A fully secure, offline-first personal workspace on your phone that pairs with your desktop and provides on-device AI automation, rich markdown organization, and smart task management.

## Requirements

### Validated

- ✓ Basic React Native / Expo foundation established.
- ✓ Custom screen and navigation architecture.
- ✓ Native elements (Custom buttons, dialogs, badges).

### Active

- [ ] **Obsidian/Notion-style Notes**: Rich Markdown note-taking with categorizable folders, tag-based organization, and clean rendering.
- [ ] **Google Tasks-style Tasks**: Fully-featured task list with lists/grouping, checklist subtasks, and due dates.
- [ ] **Birthday Tracker**: Upcoming birthdays dashboard with age calculation, remaining days, and automated local push notifications.
- [ ] **Local Wi-Fi Sync (PC App)**: Local networking sync client that scans a QR code generated by the PC app containing the local IP, port, and security token, and performs a complete merge of tasks, notes, and birthdays.
- [ ] **On-Device Local AI**: Mobile-native `llama.cpp` inference hosting a 0.5B or 1B parameter model (e.g., Qwen2.5-0.5B) to provide offline Note Summarization and Task Extraction.
- [ ] **Docker Developer Setup**: Containerized environment for local sync services, development, and system testing.

### Out of Scope

- **Cloud Databases / SaaS Accounts**: The app will remain strictly local-first and local-only over Wi-Fi to preserve privacy.
- **Third-Party AI APIs**: All AI features must run locally on-device or locally on the PC app to guarantee offline independence.

---

## Context

The project is the companion mobile app for KwestUp Desktop ([KwestUp PC Repository](https://github.com/Omprakash-p06/KwestUp.git)). The desktop app currently stores events, tasks, birthdays, and focus times in local JSON files. We are expanding the mobile app to introduce rich markdown notes and local syncing.

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Local-first Storage | Absolute privacy, offline speed, and database sync integrity. | ✓ SQLite + MMKV |
| QR Code Local Wi-Fi Sync | Simple, zero-configuration local network syncing without cloud brokers. | ✓ HTTP/WS Client |
| On-Device Local LLM | Private, zero-latency text parsing using C++ native bindings. | ✓ `react-native-llama` |
| Markdown (Obsidian-style) | Highly portable, standard text format for long-term note longevity. | ✓ Expo Markdown |
